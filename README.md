Mine Sweeper
=============

A Mine Sweeper implementation using LibGDX.

## Controls
| Control            | Description              |
| ------------------ | ------------------------ |
| Single Click       | Uncover tile             |
| Left + Right Click | Flag tile as safe        |
| R                  | Reset Game               |
| H                  | Help (Uncover safe tile) |
| Escape             | Exit Game                |

